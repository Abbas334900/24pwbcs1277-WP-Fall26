Pixels (px)

This text is 16px. Pixels are fixed units not relative to any other length.

Em Units (em)

This text is 1.25em. Em values are relative to the parent element's font size.

Root Em (rem)

This text is 1.5rem. Rem values are relative to the root (html) element's font size.

Percentage (%)

This text is 125%. Percentages are relative to the parent element's font size.

Viewport Width (vw)

This text is 4vw (responsive to viewport width).
