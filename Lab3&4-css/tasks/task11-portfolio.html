Task 11: Personal Portfolio Styling

📋 Task Overview: Apply comprehensive CSS styling to your Lab 02 portfolio project. Create a cohesive visual design with custom color scheme, improved typography, spacing, visual effects, and responsive layout.

Requirements

  1. Color Scheme: Define a cohesive color palette with primary, secondary, and accent colors
  2. Typography: Choose appropriate fonts and apply consistent sizing and spacing
  3. Layout: Use CSS Flexbox/Grid for responsive layouts
  4. Navigation: Style the navigation menu with hover effects and active states
  5. Hero Section: Create an attractive header/hero section with gradient background
  6. Cards/Sections: Style project/work cards with shadows, borders, and hover effects
  7. Images: Add image effects (border-radius, shadows, filters)
  8. Links & Buttons: Style all interactive elements with clear states
  9. Responsive Design: Ensure the portfolio works on mobile, tablet, and desktop
  10. Footer: Style footer with proper spacing and alignment

Step-by-Step Implementation

Step 1: Define Your Color Scheme

Create CSS variables for your color palette:

:root { --primary-color: #3498db; --secondary-color: #e74c3c; --accent-color: #f39c12; --text-color: #2c3e50; --light-bg: #ecf0f1; --white: #ffffff; }

Step 2: Import Google Fonts

Add font imports to your portfolio:

@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&family=Playfair+Display:wght@700&display=swap');

Step 3: Create a Modular CSS File

Name it `portfolio.css` and link it in your HTML:

<link rel="stylesheet" href="portfolio.css">

Step 4: Style Major Sections

  • Navigation bar
  • Hero/header section
  • About section
  • Portfolio/projects section
  • Skills section
  • Experience section
  • Contact/footer section

Step 5: Add Interactive Effects

  • Hover effects on navigation links
  • Hover zoom on project cards
  • Button hover states
  • Smooth transitions
  • Scroll effects (optional)

Step 6: Make It Responsive

/* Mobile First */ body { font-size: 14px; } /* Tablet */ @media (min-width: 768px) { body { font-size: 16px; } } /* Desktop */ @media (min-width: 1024px) { body { font-size: 18px; } }

CSS Properties to Apply

  • background-color and background-images
  • color for text
  • font-family, font-size, font-weight
  • padding and margin for spacing
  • border and border-radius for styling
  • box-shadow for depth
  • hover and focus states
  • display: flex or display: grid for layout
  • max-width for responsive containers
  • transition for smooth animations
  • transform for effects
  • @media queries for responsive design

💡 Styling Tips

  • Consistency: Use the same colors and fonts throughout
  • Whitespace: Don't overcrowd - use padding and margins generously
  • Contrast: Make sure text is readable against backgrounds
  • Hierarchy: Use font sizes and colors to show importance
  • Navigation: Make it clear where users can click
  • Feedback: Show hover states for interactive elements
  • Mobile First: Design for mobile, enhance for larger screens
  • Performance: Keep CSS file size reasonable

Common Issues & Solutions

Issue: Images not centered

/* Use this to center images */ img { display: block; margin: 0 auto; }

Issue: Text not readable on background

/* Add text shadow for better contrast */ h1 { color: white; text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5); }

Issue: Layout breaks on mobile

/* Use max-width for responsive containers */ .container { max-width: 1200px; margin: 0 auto; padding: 0 20px; }
