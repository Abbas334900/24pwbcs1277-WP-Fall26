Learning Outcomes

By the end of this lab, students will be able to:

  • Understand CSS syntax, rules, and selectors

  • Work with background properties and images
  • Style images and create visual effects
  • Implement CSS box model properties
  • Create gradients and shadow effects
  • Apply CSS transformations and transitions

  3. CSS Architecture: Review the main.css, typography.css, components.css, and effects.css files
  4. Customize: Modify the styles to create your own color schemes and layouts
  5. Responsive Design: Make the components work on mobile devices
  6. Practical Application: Apply these styles to your own portfolio and projects (Tasks 11-13)

💡 Tips & Best Practices

  • Use CSS Variables: Define colors and sizes in :root for easy customization
  • Mobile First: Design for mobile screens first, then enhance for larger screens
  • Performance: Use CSS transforms and transitions instead of JavaScript animations
  • Accessibility: Ensure sufficient color contrast and keyboard navigation
  • Organization: Separate styles into logical files by component
  • Browser Testing: Test in multiple browsers to ensure compatibility
  • Semantic HTML: Use proper HTML elements with CSS for better structure
  • CSS Specificity: Keep specificity low for easier maintenance
